# Fitur-fitur dan kriteria serta tambahan/opsional

Fitur-fitur : 

1. Autentikasi pengguna<br>
(Login, registrasi, lupa password — bisa menggunakan email/nomor HP atau Firebase Auth)<br>
2. Profil pengguna<br>
(Nama, foto profil, status, nomor telepon, last seen)<br>
3. Mengirim dan menerima pesan real-time<br>
(Text message)<br>
4. Mengirim file media<br>
(Foto, video, dokumen)<br>
5. List obrolan & kontak<br>
(Menampilkan daftar teman/room chat)<br>
6. Status pesan<br>
(Terkirim, terbaca — like WhatsApp: ✓ or double ✓<br>
7. Notifikasi push<br>
(Saat ada pesan baru masuk walau aplikasi tertutup)<br>
8. Update status/Story (opsional)<br>
(Foto/video teks yang hilang dalam 24 jam)<br>
9. Voice & Video call (opsional untuk tahap lanjut)<br>
(WebRTC / VoIP)<br>
10. Dark mode & UI modern<br>
(Pengalaman pengguna lebih nyaman)<br>
<br>

Kriteria-kriteria :<br>

1. Keamanan data pengguna<br>
(Enkripsi pesan, proteksi token, secure database rules)<br>
2. Real-time & Low latency<br>
(Pesan harus cepat masuk)<br>
3. UI/UX mudah digunakan (user-friendly)<br>
(Desain bersih, tombol jelas)<br>
4. Efisiensi storage & bandwidth<br>
(Compression media & cache management)<br>
5. Support minimal Android versi tertentu<br>
(misal Android 8.0 / API level 26 ke atas)<br>
6. Reliability & error handling<br>
(Tetap berjalan stabil meski koneksi lemah)<br>
7. Scalability<br>
(Siap untuk banyak pengguna dengan server cloud)<br>
8. Offline support<br>
(Pesan masih tampil tanpa internet)<br>
9. Aksesibilitas<br>
(Font yang dapat disesuaikan, screen reader friendly)<br>
10. Integrasi Backend yang aman & terstruktur<br>
(Firebase Realtime Database / Firestore / API custom)<br>
